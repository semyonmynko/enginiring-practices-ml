# Data Science Project: Iris Dataset

Проект для анализа данных на основе набора данных Iris. Проект включает в себя обработку данных, базовый анализ, построение модели классификации и оценку результатов.

## Цели проекта
1. Загрузка и обработка данных (с помощью pandas).
2. Проведение первичного анализа данных.
3. Построение модели машинного обучения для классификации.
4. Оценка модели.

## Структура проекта
- `data/` — Данные
- `notebooks/` — Jupyter notebooks для EDA
- `docs/` — Документация проекта
- `models/` — Модели машинного обучения
- `reports/` — Итоговые отчёты и графики
